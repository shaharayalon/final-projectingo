# final projectingo
 
### פרוייקט גמר - מערכת ניהול אתר

## תיאור הפרוייקט

הפרוייקט כולל פיתוח צד שרת וצד לקוח למערכת ניהול אתר המאפשרת למשתמשים עסקיים לפרסם תוכן, לערוך ולמחוק אותו, ולמשתמשים רגילים לצרוך אותו ולאהוב אותו.

### דרגי משתמשים
1. אדמין
2. משתמש עסקי
3. משתמש רגיל

## התקנה

### צד שרת
1. להוריד את הפרוייקט מתיקיית הbackend ולחלץ אותו.
2. לפתוח את הפרוייקט עם סביבת העבודה (VS Code מומלץ).
3. להתקין את כל התלויות בעזרת:
  
```bash
npm i
```

4. להגדיר את קובץ ה-`.env` בהתאם לסביבת העבודה:
   - `dev`
   - `test`
   - `prod`

5. להפעיל את הפרוייקט:
```bash
npm run watch
```
   תוצג אינדיקציה שהפרוייקט רץ בהצלחה.

### צד לקוח
1. להוריד את הפרוייקט מתיקיית הfrontend ולחלץ אותו.
2. לפתוח את הפרוייקט עם סביבת העבודה.
3. להתקין את כל התלויות בעזרת:

```bash
npm i
```

4. להפעיל את הפרוייקט:
```bash
npm start
```

5. לפתוח דפדפן ולגלוש לכתובת:
   - [http://localhost:3000](http://localhost:3000)
